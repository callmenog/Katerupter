# Katerupter

> **⚠️ Educational & Authorized Security Testing Only**

Katerupter is a wireless signal scanning and network testing toolkit created for cybersecurity education, research, and authorized penetration testing. Its purpose is to help users better understand wireless networking, signal analysis, and security testing within legal and controlled environments.

## Features

* 📡 Scan and discover nearby wireless signals
* 📶 View information about detected networks
* 🔍 Analyze wireless environments
* 🧪 Designed for security research and authorized penetration testing
* 🎓 Built as an educational cybersecurity project

## Required tools

You MUST install Python to be able to run this program. Python 3.14.6 ---> https://www.python.org/downloads/ 

## Intended Use

Katerupter is intended **only** for:

* Educational purposes
* Cybersecurity research
* Security labs
* Testing on your own equipment
* Authorized penetration tests where you have explicit permission

**Never use this software against networks or devices that you do not own or have permission to test.**
